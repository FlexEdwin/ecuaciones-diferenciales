SECCIÓN I: DERIVADAS
#1
$$f(x) = e^{3x^2 + 2x + 1}$$
Inventario de Reglas a usar:
  • 1. $\frac{d}{dx}[e^u] = e^u \cdot u'$ (Regla de la Cadena)
  • 2. $\frac{d}{dx}[x^n] = n x^{n-1}$ (Regla de la Potencia)
  • 3. $\frac{d}{dx}[cx] = c$ (Constante por variable)
  • 4. $\frac{d}{dx}[c] = 0$ (Derivada de constante)
Paso 1: Derivar el exponente ($u$) aparte. $u = 3x^2 + 2x + 1$
• Derivada de $3x^2 \to 3(2x) = 6x$
• Derivada de $2x \to 2$
• Derivada de $1 \to 0$
$u' = 6x + 2$
Paso 2: Aplicar la fórmula exponencial. $$f'(x) = e^{3x^2 + 2x + 1} \cdot (6x + 2)$$
Respuesta (Factorizando el 2):
$$f'(x) = 2(3x + 1)e^{3x^2 + 2x + 1}$$
#2
$$g(x) = \frac{x^3}{\ln(x)}$$
Inventario de Reglas:
  • 1. $\left( \frac{u}{v} \right)' = \frac{u'v - uv'}{v^2}$ (Regla del Cociente)
  • 2. $\frac{d}{dx}[x^3] = 3x^2$ (Potencia)
  • 3. $\frac{d}{dx}[\ln(x)] = \frac{1}{x}$ (Logaritmo Natural)
Paso 1: Identificar componentes. Arriba: $u = x^3 \to u' = 3x^2$
Abajo: $v = \ln(x) \to v' = \frac{1}{x}$
Paso 2: Reemplazar en la fórmula del cociente. $$g'(x) = \frac{(3x^2)(\ln x) - (x^3)(\frac{1}{x})}{(\ln x)^2}$$
Paso 3: Simplificación. Observamos que $x^3 \cdot \frac{1}{x} = x^2$. $$g'(x) = \frac{3x^2 \ln(x) - x^2}{\ln^2(x)}$$
Respuesta (Factor Común $x^2$):
$$g'(x) = \frac{x^2(3\ln(x) - 1)}{\ln^2(x)}$$
#3
$$\sin(x) + \cos(y) = xy$$
Inventario de Reglas (Muy importante):
  • 1. $\frac{d}{dx}[\sin(x)] = \cos(x)$ (Básica)
  • 2. $\frac{d}{dx}[\cos(y)] = -\sin(y) \cdot y'$ (Cadena/Implícita)
  • 3. $\frac{d}{dx}[xy] = 1\cdot y + x\cdot y'$ (Regla del Producto)
Paso 1: Derivar cada término usando la lista de arriba. $$\underbrace{\cos(x)}_{\text{De regla 1}} - \underbrace{\sin(y)y'}_{\text{De regla 2}} = \underbrace{y + xy'}_{\text{De regla 3}}$$
Paso 2: Agrupación algebraica. Objetivo: Dejar todo lo que tenga $y'$ a la izquierda. $$-\sin(y)y' - xy' = y - \cos(x)$$ *Multiplicamos por -1 para limpiar signos:* $$\sin(y)y' + xy' = \cos(x) - y$$
Paso 3: Factorizar $y'$. $$y'(\sin(y) + x) = \cos(x) - y$$
Respuesta (Despeje final):
$$y' = \frac{\cos(x) - y}{x + \sin(y)}$$
#4
$$h(x) = \ln(2x^3 - x^2 + 3x - 1)$$
Inventario de Reglas:
  • 1. $\frac{d}{dx}[\ln(u)] = \frac{u'}{u}$ (Regla Logarítmica)
  • 2. $\frac{d}{dx}[2x^3] = 6x^2$ (Potencia)
  • 3. $\frac{d}{dx}[-x^2] = -2x$ (Potencia)
  • 4. $\frac{d}{dx}[3x] = 3$ (Lineal)
Paso 1: Calcular $u'$ sumando las derivadas individuales. $u' = 6x^2 - 2x + 3$
Respuesta (Aplicando fórmula 1):
$$h'(x) = \frac{6x^2 - 2x + 3}{2x^3 - x^2 + 3x - 1}$$
#5
$$f(x) = \frac{e^x - 1}{\sqrt{x}}$$
Inventario de Reglas:
  • 1. $\frac{d}{dx}[e^x] = e^x$ (Exponencial básica)
  • 2. $\frac{d}{dx}[1] = 0$ (Constante)
  • 3. $\frac{d}{dx}[\sqrt{x}] = \frac{1}{2\sqrt{x}}$ (Derivada de raíz)
  • 4. Regla del Cociente (ver Ej. 2)
Paso 1: Armar componentes. $u = e^x - 1 \to u' = e^x$
$v = \sqrt{x} \to v' = \frac{1}{2\sqrt{x}}$
Paso 2: Aplicar Cociente. $$f'(x) = \frac{(e^x)(\sqrt{x}) - (e^x - 1)(\frac{1}{2\sqrt{x}})}{(\sqrt{x})^2}$$
Paso 3: Limpieza (Racionalización interna). Multiplicamos todo por $2\sqrt{x}$ para eliminar la fracción pequeña. $$f'(x) = \frac{2x e^x - (e^x - 1)}{2x\sqrt{x}}$$
Respuesta:
$$f'(x) = \frac{e^x(2x - 1) + 1}{2x\sqrt{x}}$$
#6
$$g(x) = \frac{x^2 \sin(x)}{x^2 + 1}$$
Inventario de Reglas:
  • 1. $(uv)' = u'v + uv'$ (Producto para numerador)
  • 2. $\frac{d}{dx}[\sin x] = \cos x$ (Trigonométrica)
  • 3. $\frac{d}{dx}[x^2] = 2x$ (Potencia)
  • 4. Regla del Cociente (para la función total)
Paso 1: Derivar Numerador (Ojo: Es un producto). Derivada de $x^2 \sin(x)$: $$(2x)(\sin x) + (x^2)(\cos x)$$
Paso 2: Armar el Cociente gigante. $$g'(x) = \frac{[2x\sin x + x^2\cos x](x^2+1) - (x^2\sin x)(2x)}{(x^2+1)^2}$$
Paso 3: Simplificar. Al multiplicar, el término $2x^3\sin x$ se cancela con el $-2x^3\sin x$ del final.
Respuesta:
$$g'(x) = \frac{x^2(x^2+1)\cos x + 2x\sin x}{(x^2+1)^2}$$
#7
$$x^3 + y^3 = 9xy$$
Inventario de Reglas:
  • 1. $\frac{d}{dx}[x^3] = 3x^2$
  • 2. $\frac{d}{dx}[y^3] = 3y^2 \cdot y'$ (Implícita)
  • 3. $\frac{d}{dx}[9xy] = 9(y + xy')$ (Constante por Producto)
Paso 1: Derivar usando las reglas listadas. $$3x^2 + 3y^2y' = 9y + 9xy'$$
Paso 2: Dividir por 3 (Simplificación). $$x^2 + y^2y' = 3y + 3xy'$$
Paso 3: Agrupar $y'$. $$y^2y' - 3xy' = 3y - x^2$$ $$y'(y^2 - 3x) = 3y - x^2$$
Respuesta:
$$y' = \frac{3y - x^2}{y^2 - 3x}$$
#8
$$h(x) = \frac{\ln(x)}{x}$$
Inventario de Reglas:
  • 1. $\ln(x) \to 1/x$
  • 2. $x \to 1$
  • 3. Regla del Cociente
Sustitución en fórmula: $$h'(x) = \frac{(\frac{1}{x})(x) - (\ln x)(1)}{x^2}$$
Respuesta (Recordando que $x/x = 1$):
$$h'(x) = \frac{1 - \ln(x)}{x^2}$$
#9
$$f(x) = e^{x^3}$$
Inventario de Reglas:
  • 1. $\frac{d}{dx}[e^u] = e^u \cdot u'$ (Cadena)
  • 2. $\frac{d}{dx}[x^3] = 3x^2$ (Potencia)
Cálculo: Derivada interna: $3x^2$.
Multiplicamos función original por interna.
Respuesta:
$$f'(x) = 3x^2 e^{x^3}$$
#10
$$g(x) = \ln(x) \cdot \cos(x)$$
Inventario de Reglas:
  • 1. Regla del Producto: $u'v + uv'$
  • 2. Derivada $\ln x = 1/x$
  • 3. Derivada $\cos x = -\sin x$
Aplicación: $$g'(x) = (\frac{1}{x})\cos(x) + \ln(x)(-\sin x)$$
Respuesta:
$$g'(x) = \frac{\cos(x)}{x} - \ln(x)\sin(x)$$
#11
$$e^x + y^2 = \sin(xy)$$
Inventario de Reglas:
  • 1. $\frac{d}{dx}[e^x] = e^x$
  • 2. $\frac{d}{dx}[y^2] = 2y \cdot y'$ (Implícita)
  • 3. $\frac{d}{dx}[\sin(u)] = \cos(u) \cdot u'$ (Cadena)
  • 4. $\frac{d}{dx}[xy] = y + xy'$ (Producto)
Paso 1: Derivar ambos lados. $$e^x + 2yy' = \cos(xy) \cdot (y + xy')$$
Paso 2: Distribuir (Romper paréntesis). $$e^x + 2yy' = y\cos(xy) + xy'\cos(xy)$$
Paso 3: Despejar $y'$. $$y'(2y - x\cos(xy)) = y\cos(xy) - e^x$$
Respuesta:
$$y' = \frac{y\cos(xy) - e^x}{2y - x\cos(xy)}$$
#12
$$h(x) = \frac{\sin(x^2)}{x}$$
Inventario de Reglas:
  • 1. $\frac{d}{dx}[\sin(x^2)] = \cos(x^2) \cdot 2x$ (Cadena)
  • 2. $\frac{d}{dx}[x] = 1$
  • 3. Regla del Cociente
Sustitución en Cociente: $$h'(x) = \frac{[2x\cos(x^2)](x) - \sin(x^2)(1)}{x^2}$$
Respuesta:
$$h'(x) = \frac{2x^2\cos(x^2) - \sin(x^2)}{x^2}$$
#13
$$f(x) = \sqrt{e^{2x} - 1}$$
Inventario (Orden de cebolla):
  • 1. Raíz: $\frac{1}{2\sqrt{u}}$
  • 2. Exponencial: $e^{2x}$
  • 3. Lineal: Derivada de $2x$ es $2$
Multiplicación en Cadena: $$f'(x) = \frac{1}{2\sqrt{e^{2x}-1}} \cdot e^{2x} \cdot 2$$
Respuesta (Cancelando los 2):
$$f'(x) = \frac{e^{2x}}{\sqrt{e^{2x} - 1}}$$
#14
$$g(x) = \frac{\tan(x)}{x^2}$$
Inventario de Reglas:
  • 1. $\tan(x) \to \sec^2(x)$
  • 2. $x^2 \to 2x$
  • 3. Regla del Cociente
Aplicación: $$g'(x) = \frac{\sec^2(x) \cdot x^2 - \tan(x) \cdot 2x}{x^4}$$
Respuesta (Simplificando una $x$):
$$g'(x) = \frac{x\sec^2(x) - 2\tan(x)}{x^3}$$
#15
$$x^4 + y^4 = 16$$
Inventario:
  • 1. $x^4 \to 4x^3$
  • 2. $y^4 \to 4y^3 y'$
  • 3. Constante $16 \to 0$
Desarrollo: $4x^3 + 4y^3y' = 0$
$4y^3y' = -4x^3$
Respuesta:
$$y' = -\frac{x^3}{y^3}$$
SECCIÓN II: INTEGRALES
#1
$$I = \int e^{x^2} \sin(x) \, dx$$
Intento: Sustitución Si $u = x^2 \Rightarrow du = 2x dx$
Análisis:
Al sustituir, nos sobra un término $\sin(x)$ y falta una $x$ en el diferencial. No se puede resolver. Integral No Elemental.
#2
$$I = \int \frac{1}{x \ln(x)} \, dx$$
Herramientas:
  • 1. Derivada de $\ln x = 1/x$ (Esta es la clave)
  • 2. $\int \frac{1}{u} du = \ln|u|$
Sustitución: $u = \ln x \to du = \frac{1}{x} dx$.
La integral se convierte exactamente en $\int \frac{1}{u} du$.
Respuesta:
$$I = \ln|\ln(x)| + C$$
#3
$$I = \int \cos^2(x) \, dx$$
Herramienta Principal: Identidad de Reducción: $\cos^2 x = \frac{1 + \cos(2x)}{2}$
Separación: $$I = \int \frac{1}{2} dx + \int \frac{1}{2}\cos(2x) dx$$
Respuesta:
$$I = \frac{x}{2} + \frac{\sin(2x)}{4} + C$$
#4
$$I = \int \frac{\ln(x)}{\sqrt{x}} \, dx$$
Herramientas:
  • 1. Sustitución $w = \sqrt{x}$
  • 2. Integración por Partes $\int \ln w$
Desarrollo: Al sustituir, llegamos a $4\int \ln w dw$.
Sabemos que $\int \ln w = w\ln w - w$.
Respuesta:
$$I = 4\sqrt{x}\ln(\sqrt{x}) - 4\sqrt{x} + C$$
#5
$$I = \int \frac{e^x}{x^2 + 1} \, dx$$
Falla la sustitución $u=x^2+1$ (falta $2x$) y $u=e^x$. No elemental.
#6
$$I = \int \frac{x \sin(x)}{x^2 + 1} \, dx$$
La integración por partes complica el término algebraico con logaritmos. No elemental.
#7
$$I = \int x^2 \cos(x) \, dx$$
Herramienta: Partes x2 $\int u dv = uv - \int v du$. Usar dos veces para bajar el grado de $x^2$.
Iteraciones: 1. $x^2 \to 2x$
2. $2x \to 2$
Integrando $\cos \to \sin \to -\cos$.
Respuesta:
$$I = x^2\sin(x) + 2x\cos(x) - 2\sin(x) + C$$

#14
$$I = \int x^2 e^x \, dx$$

Armado: $x^2e^x - 2xe^x + 2e^x$.
Respuesta:
$$I = e^x(x^2 - 2x + 2) + C$$
#15
$$I = \int \frac{1}{x^2 + 4x + 5} \, dx$$
Herramienta: Completar Cuadrado $$x^2 + bx + c = (x + b/2)^2 + (c - (b/2)^2)$$
Transformación: $x^2 + 4x + 5 = (x+2)^2 + 1$. Integral $\int \frac{1}{u^2+1}$.
Respuesta:
$$I = \arctan(x + 2) + C$$
