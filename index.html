I. DERIVADAS
#1
$$f(x) = e^{3x^2 + 2x + 1}$$
Recordatorio: Derivada de $e^u$ es $e^u \cdot u'$.
¡No olvidar derivar el exponente!
1. Derivando el exponente ($u$) aparte: $u = 3x^2 + 2x + 1$
$u' = (3 \cdot 2x) + (2 \cdot 1) + \cancel{0}$
$u' = 6x + 2$
2. Multiplicando: $f'(x) = e^{3x^2 + 2x + 1} \cdot (6x + 2)$
Respuesta:
$$f'(x) = 2(3x + 1)e^{3x^2 + 2x + 1}$$
#2
$$g(x) = \frac{x^3}{\ln(x)}$$
Fórmula Cociente: (Derivada Arriba $\cdot$ Abajo) - (Arriba $\cdot$ Derivada Abajo) / (Abajo)$^2$
1. Preparando piezas: Arriba: $3x^2$
Abajo (derivado): $1/x$
2. Armado: $$g'(x) = \frac{(3x^2)(\ln x) - (x^3)(1/x)}{(\ln x)^2}$$
3. Simplificando el término derecho: $x^3 \cdot \frac{1}{x} = x^2$
Respuesta:
$$g'(x) = \frac{x^2(3\ln(x) - 1)}{\ln^2(x)}$$
#3
$$\sin(x) + \cos(y) = xy$$
1. Derivando todo (Ojo con la Implícita): $\cos(x) - \sin(y)y' = (1)y + x(y')$
2. Pasando $y'$ a un lado: $- \sin(y)y' - xy' = y - \cos(x)$
(Multiplico por -1)
$\sin(y)y' + xy' = \cos(x) - y$
Respuesta:
$$y' = \frac{\cos(x) - y}{x + \sin(y)}$$
#4
$$h(x) = \ln(2x^3 - x^2 + 3x - 1)$$
Regla Logaritmo: Derivada de adentro / Lo de adentro
Derivada interna: $6x^2 - 2x + 3$
Respuesta:
$$h'(x) = \frac{6x^2 - 2x + 3}{2x^3 - x^2 + 3x - 1}$$
#5
$$f(x) = \frac{e^x - 1}{\sqrt{x}}$$
1. Derivada del denominador ($\sqrt{x}$): $\frac{1}{2\sqrt{x}}$
2. Fórmula del cociente: $$f'(x) = \frac{e^x\sqrt{x} - (e^x - 1)\frac{1}{2\sqrt{x}}}{x}$$
3. Limpieza algebraica: Multiplico todo por $2\sqrt{x}$ para quitar esa fracción fea de arriba. $$2xe^x - (e^x - 1)$$
Respuesta:
$$f'(x) = \frac{e^x(2x - 1) + 1}{2x\sqrt{x}}$$
#6
$$g(x) = \frac{x^2 \sin(x)}{x^2 + 1}$$
¡Cuidado! El numerador es un producto. Primero derivo eso.
1. Derivada del Numerador: $2x\sin(x) + x^2\cos(x)$
2. Cociente completo: $$g'(x) = \frac{[2x\sin x + x^2\cos x](x^2+1) - (x^2\sin x)(2x)}{(x^2+1)^2}$$
3. Cancelación: $2x^3\sin(x)$ se cancela con $-2x^3\sin(x)$.
Respuesta:
$$g'(x) = \frac{x^2(x^2+1)\cos x + 2x\sin x}{(x^2+1)^2}$$
#7
$$x^3 + y^3 = 9xy$$
1. Derivando (Divido por 3 de una vez): $x^2 + y^2y' = 3y + 3xy'$
2. Agrupando términos $y'$: $y^2y' - 3xy' = 3y - x^2$
Respuesta:
$$y' = \frac{3y - x^2}{y^2 - 3x}$$
#8
$$h(x) = \frac{\ln(x)}{x}$$
Aplicación Directa: $$h'(x) = \frac{(1/x)(x) - \ln(x)}{x^2}$$ El $1/x \cdot x$ se vuelve 1.
Respuesta:
$$h'(x) = \frac{1 - \ln(x)}{x^2}$$
#9
$$f(x) = e^{x^3}$$
Derivada interna de $x^3$: $3x^2$
Respuesta:
$$f'(x) = 3x^2 e^{x^3}$$
#10
$$g(x) = \ln(x) \cdot \cos(x)$$
Regla del Producto: $u'v + uv'$
Desarrollo: $(\frac{1}{x})\cos(x) + \ln(x)(-\sin x)$
Respuesta:
$$g'(x) = \frac{\cos(x)}{x} - \ln(x)\sin(x)$$
#11
$$e^x + y^2 = \sin(xy)$$
1. Derivando el lado derecho (Cadena): $\cos(xy) \cdot (y + xy')$
2. Ecuación completa: $e^x + 2yy' = y\cos(xy) + xy'\cos(xy)$
3. Despeje: $2yy' - xy'\cos(xy) = y\cos(xy) - e^x$
Respuesta:
$$y' = \frac{y\cos(xy) - e^x}{2y - x\cos(xy)}$$
#12
$$h(x) = \frac{\sin(x^2)}{x}$$
Ojo: La derivada de $\sin(x^2)$ es $\cos(x^2) \cdot 2x$.
Cociente: $$h'(x) = \frac{[2x\cos(x^2)](x) - \sin(x^2)}{x^2}$$
Respuesta:
$$h'(x) = \frac{2x^2\cos(x^2) - \sin(x^2)}{x^2}$$
#13
$$f(x) = \sqrt{e^{2x} - 1}$$
Cadena triple (Raíz $\to$ Exponencial $\to$ 2x): $\frac{1}{2\sqrt{e^{2x}-1}} \cdot e^{2x} \cdot 2$
Cancelación: El 2 multiplica y divide. Se va.
Respuesta:
$$f'(x) = \frac{e^{2x}}{\sqrt{e^{2x} - 1}}$$
#14
$$g(x) = \frac{\tan(x)}{x^2}$$
1. Derivando: $\tan x \to \sec^2 x$
$x^2 \to 2x$
2. Fórmula: $$g'(x) = \frac{\sec^2(x) \cdot x^2 - \tan(x) \cdot 2x}{x^4}$$
3. Factorizar $x$: Arriba hay $x$ en ambos lados. Saco factor común y mato una de abajo.
Respuesta:
$$g'(x) = \frac{x\sec^2(x) - 2\tan(x)}{x^3}$$
#15
$$x^4 + y^4 = 16$$
1. Derivada rápida: $4x^3 + 4y^3y' = 0$
2. Despeje: $y' = \frac{-4x^3}{4y^3}$
Respuesta:
$$y' = -\frac{x^3}{y^3}$$
II. INTEGRALES
#1
$$I = \int e^{x^2} \sin(x) \, dx$$
Intento de Sustitución:
$$u = x^2 \Rightarrow dx = \frac{du}{2x}$$ $$\int e^u \sin(x) \frac{du}{2x}$$
Conclusión: No se puede eliminar la $x$ ni la $\sin(x)$. La integral se traba aquí. No tiene solución elemental.
#2
$$I = \int \frac{1}{x \ln(x)} \, dx$$
Sustitución ($u = \ln x$): $du = \frac{1}{x} dx$.
La integral queda limpia: $\int \frac{1}{u} du$.
Respuesta:
$$I = \ln|\ln(x)| + C$$
#3
$$I = \int \cos^2(x) \, dx$$
Identidad trigonométrica: $\cos^2 x = \frac{1 + \cos(2x)}{2}$
Integrando partes: $\int 1/2 \to x/2$
$\int \cos(2x) \to \frac{1}{2}\sin(2x)$
Respuesta:
$$I = \frac{x}{2} + \frac{\sin(2x)}{4} + C$$
#4
$$I = \int \frac{\ln(x)}{\sqrt{x}} \, dx$$
1. Quitar raíz ($w=\sqrt{x}$): $dx = 2w dw$. Integral $\to 4\int \ln w dw$.
2. Resultado conocido de $\ln w$: $w\ln w - w$.
Respuesta:
$$I = 4\sqrt{x}\ln(\sqrt{x}) - 4\sqrt{x} + C$$
#5
$$I = \int \frac{e^x}{x^2 + 1} \, dx$$
Análisis: Falla sustitución $u=x^2+1$ (falta $2x$) y $u=e^x$ (queda logaritmo abajo). No Elemental.
#6
$$I = \int \frac{x \sin(x)}{x^2 + 1} \, dx$$
Análisis: Partes genera $\int \ln(x^2+1)\cos x$. Más difícil que la original.
#7
$$I = \int x^2 \cos(x) \, dx$$
Doble Partes (Para matar el $x^2$): 1. $x^2\sin x - \int 2x\sin x$
2. $-2x\cos x + \int 2\cos x$
Respuesta:
$$I = x^2\sin(x) + 2x\cos(x) - 2\sin(x) + C$$
#8
$$I = \int \ln(x) \sin(x) \, dx$$
Llega a $\int \frac{\cos x}{x} dx$. Esto es $Ci(x)$ (Integral Coseno). No elemental.
#9
$$I = \int \frac{e^x}{x^3} \, dx$$
Llega a $\int \frac{e^x}{x} dx$. Esto es $Ei(x)$ (Integral Exponencial).
#10
$$I = \int \frac{\cos(x)}{\sin(x)} \, dx$$
Observación: Arriba tengo la derivada de lo de abajo ($du/u$).
Respuesta:
$$I = \ln|\sin(x)| + C$$
#11
$$I = \int x^2 e^{-x^2} \, dx$$
Relacionada con la Función Error (Gaussiana). No elemental.
#12
$$I = \int \ln(x) \cos(x) \, dx$$
Llega a $Si(x)$ (Integral Seno). No elemental.
#13
$$I = \int \frac{e^x}{x^2 \ln(x)} \, dx$$
Mezcla trascendente sin solución analítica.
#14
$$I = \int x^2 e^x \, dx$$
Método Tabular: Derivar: $x^2 \to 2x \to 2 \to 0$
Integrar: $e^x \to e^x \to e^x \to e^x$
Unir diagonales (+ - +): $x^2e^x - 2xe^x + 2e^x$
Respuesta:
$$I = e^x(x^2 - 2x + 2) + C$$
#15
$$I = \int \frac{1}{x^2 + 4x + 5} \, dx$$
Completar el cuadrado: $x^2 + 4x + 5 = (x+2)^2 + 1$
Integral Arco Tangente: $\int \frac{1}{u^2+1} du = \arctan(u)$
Respuesta:
$$I = \arctan(x + 2) + C$$
